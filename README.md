## Project Overview
[cite_start]Sensors, a Gateway, and a Network/Cloud-Server typically make up a conventional LoRaWAN Network[cite: 30]. [cite_start]However, in use cases like monitoring the temperature inside a truck, an interruption in external communication makes it impossible to adjust cooling set points[cite: 43, 45]. 

[cite_start]The goal of this project is to develop an Intelligent LoRa Gateway based on a Raspberry Pi to make local decisions and control systems directly[cite: 46]. [cite_start]By utilizing our own gateway as a domain controller, we protect incoming data locally and eliminate the dependency on external network servers like The Things Network (TTN)[cite: 307, 311, 312].

## Architecture & Data Flow
[cite_start]The data flow tracks how information is generated by an end device and carried through the network[cite: 483]:
1. [cite_start]**Sensing:** The RN2483 LoRa Motes utilize onboard light and temperature sensors to generate data[cite: 466].
2. [cite_start]**Transmission:** The RN2483 encrypts the data and transmits it via the LoRa network[cite: 485]. 
3. [cite_start]**Gateway Reception:** The Raspberry Pi-based LoRaWAN Gateway receives the radio packet and forwards it to a local UDP server[cite: 485, 516].
4. [cite_start]**Decoding:** A local Node.js UDP server receives the payload in Base64 format[cite: 445]. [cite_start]The JavaScript application decodes the Base64 data into HEX decimal, verifies the Message Integrity Code (MIC), and decrypts the physical payload into readable ASCII text[cite: 445, 964, 1055].

## Hardware Used
* [cite_start]**Microchip RN2483 LoRa Technology Transceiver Module:** A low-power, long-range LoRa module used as the end-node (Motes)[cite: 239, 298].
* [cite_start]**Raspberry Pi 3B+:** The core single-board computer running the Raspbian OS[cite: 276, 823].
* [cite_start]**LRWCCx-MPCIE Concentrator Card:** A micro PCIe-based LoRaWAN concentrator card based on Semtech's SX1301 or SX1308 chipset[cite: 278].
* [cite_start]**SBCPOE RPi Hat:** Provides PoE power to the Raspberry Pi and houses the concentrator card[cite: 281, 283].

## Software & Dependencies
